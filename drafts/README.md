# Drafts Directory

*General-purpose drafts that don't fit other categories.*

---

## Structure

```
drafts/
├── README.md (this file)
├── ideas/     — Half-formed thoughts, concepts
├── working/   — Active drafts
└── archive/   — Old drafts (keep for reference)
```

## When to Use

- Something that doesn't fit content/, consulting/, or crm/
- Early-stage ideas that might not go anywhere
- Scratch drafts before you know where they belong

## Naming

`[topic]-[date]-[status].md`

Example: `token-launch-2026-02-idea.md`

---

*Archive drafts older than 30 days if not actively worked.*
